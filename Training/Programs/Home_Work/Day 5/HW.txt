### Category 1: Variables, Syntax & Data Types
1. Write a Python script that assigns the values "Apple", "Banana", and "Cherry" to three different variables in a single line using unpacking.
2. Create a global variable `x = 50`. Write a function that uses the `global` keyword to change `x` to `100`, and print `x` before and after calling the function.
3. Write a program that takes a string `"150"`, casts it to an integer, adds `50`, and then casts the result back to a float.
4. Demonstrate how to print "Hello" and "World" on the same line, separated by a hyphen `-`, using the `end` parameter in the `print()` function.
5. Create a variable with the complex number `5+3j`. Extract and print its data type using the `type()` function.
6. Write a script that uses a multiline string (`"""..."""`) purely as a comment to explain the code below it.
7. Assign the same string value `"Python"` to three variables (`a`, `b`, `c`) simultaneously using a single line of code.
8. Use the `random` module to generate and print a random integer between 1 and 1000.
9. Write a script that assigns a number using scientific `E` notation (e.g., representing 45,000) and prints it out as a standard float.
10. Identify the invalid variable names in this list and explain why: `1st_name`, `_last_name`, `Total-Score`, `myVar2`.

14. Use an f-string to print the sentence "The total price is $X", where X is the variable `price = 49.998`, formatted to exactly two decimal places.
15. Use the `.replace()` method to change all instances of the letter "a" to "@" in the string "macadamia".
16. Write a program that splits the string `csv = "apple,banana,cherry"` into a list using the comma as a separator.
17. Check if the string `code = "7890"` consists purely of numbers using a built-in string method.
18. Use escape characters to correctly print this exact output on two lines: 
    Line 1: "It's a beautiful day!"
    Line 2: \Let's code\
19. Write a script using the `.zfill()` method to format the number `7` so it outputs as `"0007"`.

### Category 3: Operators & Boolean Logic
21. Write a script to check if a variable `num = 21` is completely divisible by both 3 and 7 using the modulus operator (`%`) and the logical `and` operator.
22. Calculate `(10 + 5) * 2 ** 3 / 4`. Manually figure out the order of precedence and write the code to confirm the final result.
23. Start with `x = 10`. Use shorthand assignment operators to multiply it by 5, subtract 10, and then perform floor division by 2. Print the final answer.
24. Create two lists: `l1 = [1, 2, 3]` and `l2 = [1, 2, 3]`. Print the boolean results of `l1 == l2` and `l1 is l2`. 
25. Perform a bitwise AND (`&`) operation on the integers 14 and 9. Print the result.
26. Write an expression that evaluates to `True` if `x` is between 50 and 100 (inclusive) using chained comparison operators (e.g., `50 <= x...`).
27. Use the `ord()` function to find the ASCII value of the letter 'Z'.
28. Evaluate and print `bool(0.0)`, `bool(" ")`, and `bool(None)`. 
29. Perform a bitwise left shift (`<<`) by 2 on the number 5 and print the result.
30. Write an `if/else` statement using logical operators that prints "Access Granted" if `age >= 18` OR `has_permission == True`, but NOT if both are true (simulate XOR logic).

39. Clear all the items from `data = [1, 2, 3]` using a list method, and print the resulting empty list.
